# 🧹 Data Cleaning using Python

A beginner-friendly implementation of **Data Cleaning** using **Python**, **Pandas**, and **NumPy**. This project demonstrates essential data preprocessing techniques such as handling missing values, removing duplicate records, correcting inconsistent data, and preparing datasets for Machine Learning and Data Analysis.

---

# 📌 Project Overview

This project demonstrates the complete **Data Cleaning** process used in real-world data science projects. It focuses on improving data quality by identifying and resolving common issues before performing analysis or building machine learning models.

The notebook is designed for **Google Colab** and is suitable for beginners learning **Data Science**, **Machine Learning**, and **Data Preprocessing**.


Move into the project folder

```bash
cd data-cleaning
```

Install the required libraries

```bash
pip install pandas numpy
```
